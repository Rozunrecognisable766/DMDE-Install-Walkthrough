## 🔍 Solving Typical Errors

Users sometimes face issues when they try to run the tool. Follow these fixes if the program fails.

### Windows 11 Compatibility
Windows 11 includes strict security features. If the software seems to freeze, right-click the dmde.exe file. Select Properties. Check the box that says Run this program in compatibility mode for Windows 8. Click Apply and then OK.

### Administrator Rights
DMDE requires full access to your disk hardware. If you do not run it as an administrator, it cannot see your drives. Right-click the dmde.exe file. Choose Run as Administrator. Click Yes on the security prompt.

### Anti-Virus Blocks
Some anti-virus programs flag recovery tools as threats. If the file disappears after you extract it, check your Anti-virus Quarantine folder. You may need to add an exception for the DMDE folder in your settings.

## ⚠️ Important Safety Rules

Never recover files onto the same drive where you lost data. Writing new data to that drive destroys the files you want to save. Always use an external hard drive or a USB thumb drive as the destination for your recovered data.

If you hear clicking noises from your hard drive, stop your computer at once. Physical damage requires a professional lab. Software cannot fix hardware parts.
